# ✨ ElevateEcom - Luxury E-commerce Agency Website

A premium black & gold themed e-commerce agency website with glassmorphism effects and smooth animations.

## 🚀 Features
- Sticky navigation bar
- Hero section with dashboard mockup
- Trusted platforms showcase (Amazon, Shopify, etc.)
- Services section
- Animated statistics counter
- Testimonials slider
- FAQ accordion
- Contact form
- WhatsApp floating button
